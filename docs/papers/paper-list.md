# 脑胶质瘤MRI检测核心文献（均含开源代码）
## 1. 3D U-Net: Learning Dense Volumetric Segmentation from Sparse Annotation
- 会议：MICCAI 2016（医学影像顶会）
- 代码：https://github.com/wolny/pytorch-3dunet
- 贡献：3D医学图像分割经典基线，脑胶质瘤分割开山之作

## 2. nnU-Net: a self-configuring method for deep learning-based biomedical image segmentation
- 期刊：Nature Methods 2021（顶刊，IF≈47）
- 代码：https://github.com/MIC-DKFZ/nnUNet
- 贡献：全自动医学图像分割框架，BraTS挑战赛持续SOTA

## 3. TransBTS: Multimodal Brain Tumor Segmentation Using Transformer
- 会议：MICCAI 2021
- 代码：https://github.com/Wenxuan-1997/TransBTS
- 贡献：Transformer+3D CNN，捕捉长距离依赖，提升分割精度

## 4. BraTS 2020: Multimodal Brain Tumor Segmentation Challenge
- 会议：MICCAI BraTS 2020
- 代码：https://github.com/NeuroAI-lab/BraTS2020
- 贡献：脑胶质瘤分割基准数据集，提供多模态MRI数据

## 5. DeepMedic: A Fast 3D CNN for Brain Tumor Segmentation
- 期刊：IEEE TMI 2016（顶刊）
- 代码：https://github.com/Kamnitsask/deepmedic
- 贡献：轻量级3D CNN，适合临床实时检测
